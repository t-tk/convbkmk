convbkmk samples


The files in this directory are
samples for test of convbkmk.


Makefile  : Makefile for Unix-like OS
check.sh  : check script for convbkmk function
test/*    : temporary files
output/*  : output results

bkmk-dpps.tex
  : test for (u)platex + dvips.

bkmk-dpdx.tex, bkmk-docinfo.txt
  : test for (u)platex + dvipdfmx.
